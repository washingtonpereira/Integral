# Integral

Projeto para uma loja de produtos naturais ficticia, onde os clientes conseguem fazer consultas de produtos no qual a loja tém e o valor respectivo do Kg de cada produto.
-------------
### Tecnologias Utilizadas 
-   Python 3.11.4
-   Django 5.0.2
-   Bootstrap v5
-   Sqlite3
